Class Hierarchy

  • telesocial.RichClient: More Object Oriented wrapper around SimpleClient.
  • telesocial.RichClientItem: Base class for API items like Media, Conference or NetworkID
    • telesocial.Conference: Conference item, storing conference_id and providing convenient access to conference related methods.
    • telesocial.Media: Media item, storing media_id and providing convenient access to media related methods and properties.
    • telesocial.NetworkId: NetworkId item, storing network_id and providing convenient access to network_id related methods.
  • telesocial.SimpleClient: Provides basic interface to Telesocial REST API.
  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • telesocial.TelesocialError: Base class for all errors originating from telesocial module.
          • telesocial.TelesocialNetworkError
          • telesocial.TelesocialServiceError
    • tuple: tuple() -> empty tuple tuple(iterable) -> tuple initialized from iterable's items
      • telesocial.Response: Response(code, data)
